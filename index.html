SecureVault
Zero-knowledge password manager
↗ GitHub ⚡ Features

Keep passwords encrypted—even from yourself.

SecureVault encrypts your vault with strong, modern cryptography. The master password never gets stored; everything is encrypted client-side.

Breach checking via HIBP k-anonymity (SHA-1 prefix only)
View the code How it works CLI commands

Built for safety and usability

  • Argon2id key derivation
  • AES-256-GCM authenticated encryption
  • AES-GCM vault file (single encrypted vault: vault.enc)
  • Typer + Rich CLI with search, generator, breach check, export/import, and auto-lock
🔐
Zero-knowledge encryption
Master password is never stored. Vault contents are encrypted before writing to disk.
🧠
Breach checking
HIBP k-anonymity: send only SHA-1 prefixes; compare counts locally.
🧪
Strength scoring
Password strength evaluation using zxcvbn.
🗄️
Export/Import
Move encrypted data via JSON without breaking zero-knowledge guarantees.
